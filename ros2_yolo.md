Instructions regarding the setup of the YOLOv8 object detection model by Ultralytics for any camera in Isaac Sim.

**yolo_ros AND YOLO v8 INSTALLATION**

1. Install yolo_ros from the following repo by following the installation instructions: https://github.com/mgonzs13/yolo_ros/tree/main

   If not possible, you can first create a virtual environment and install it on there by first entering the following in a terminal and the following the rest of the instructions on the yolo_ros github:
   cd ~/ros2_ws/src  
  python3 -m venv venv  
  source venv/bin/activate  
   
2. Install Ultralytics YOLO v8: https://docs.ultralytics.com/quickstart#headless-server-installation
